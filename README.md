# garudaku-frontend-test
## Description
repo for garudaku frontend developer test
## Requirements
```
- Node v12.x
- Yarn Pkg Manager
- Vue 2.x / NuxtJs
- Vuex State mgmt
- SCSS
- Atomic design
```
